# Ponteiros

### Minha versão

De acordo com o aprendido uma ponteiro é uma variável que aponta para algo, este algo é o endereÃ§o de memória, ou seja o ponteiro aponta para o endereço de memória de uma variável, isto possibilita que você "brinque" com a memória em seu programa.

- Exemplo:
   int n = 2;
   int *p;
   p = &n;
   std::cout << *p << std::endl;   //será impresso o valor 2;

   - Nesse exemplo, o ponteiro `p`, recebe o endereço de memória da variável `n`, assim o conteúdo de `p` passa a ser o mesmo de n, pois `p` está apontando para o endereço de `n`, portanto quando executar o programa o valor de `*p` será 2. O `*` Ã© para acessar o conteÃºdo de `p`.
